REQUIREMENTS:
-----------------------------
JSON API written in Node.js / Express / CoffeeScript (the technologies we currently use). The API should provide endpoints to accomplish the following:
- Create a new user (username, password, first, last, etc)
- Response should return a unique id for the user
- Sign in the user (via username and pass)
- Response should return a token
- Fetch, update, and delete user data
- Return a sensible response for each method


BOOTSTRAP:
-----------------------------
- Node JS installed
- Mongo DB installed and running with 'test' DB, port: 27017
- In terminal run 'nmp install' to install dependencies
- In terminal run 'npm start'
- Browse to http://localhost:8080/
